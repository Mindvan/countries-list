# Исследование стран

Веб-приложение для просмотра информации о странах с возможностью поиска, фильтрации и добавления в избранное. Приложение получает данные с внешнего API [Rest Countries](https://restcountries.com/) и оптимизировано для работы с большим списком элементов.

Используется:
- React 19;
- React Router DOM 7;
- TypeScript;
- Styled-Components;
- Vite;
- ESLint 9 + typescript-eslint.

Деплой: https://mindvan.github.io/countries/
